# Pasta de Áudios MP3 do Treinador - ApenasCorrer

Coloque aqui os arquivos de áudio gravados em formato `.mp3` para substituir a voz sintetizada padrão do aplicativo. 

Se o aplicativo não encontrar um arquivo correspondente, ele usará a **síntese de voz (TTS)** do celular automaticamente como plano de fundo reserva.

### Lista de Arquivos Esperados:

| Nome do Arquivo | Quando é Reproduzido | Exemplo de Fala Recomendada |
| :--- | :--- | :--- |
| **`aquecimento.mp3`** | Início do treino (Aquecimento) | *"Atenção, vamos começar com o aquecimento. Caminhada leve, mantenha os braços relaxados."* |
| **`caminhar.mp3`** | Transição para Caminhada | *"Caminhada moderada agora. Aproveite para recuperar o fôlego."* |
| **`correr.mp3`** | Transição para Corrida (Trote) | *"Hora de correr! Ritmo leve, não force o passo, respire fundo."* |
| **`desafio.mp3`** | Desafios Especiais (2km, 3km, 5km) | *"Atenção! Começa o seu desafio de corrida contínua. Mantenha o foco, controle a respiração e vá até o fim!"* |
| **`desaquecimento.mp3`** | Fim do treino (Desaquecimento) | *"Treino concluído. Hora de desaquecer com caminhada lenta."* |
| **`alongar.mp3`** | Fim absoluto do treino (Alongamento) | *"Excelente! Faça alguns alongamentos leves para pernas e panturrilhas."* |
| **`fim.mp3`** | Ao concluir o treino completo | *"Parabéns! Você completou com sucesso o treino de hoje! Muito bom trabalho!"* |
| **`payment_success.mp3`**| Confirmação de compra do Premium | *"Parabéns! Seu pagamento foi confirmado. Agora você é um membro premium do Apenas Correr! Todos os treinos foram liberados."* |

---

### Dica para Gerar Áudios Humildes e Profissionais:
Você pode gerar esses arquivos gratuitamente com vozes extremamente reais e humanas (em português do Brasil) usando ferramentas de IA como:
1. **ElevenLabs** (elevenlabs.io)
2. **Play.ht** (play.ht)

Basta digitar os textos, escolher a voz mais amigável, fazer o download dos arquivos MP3, renomeá-los conforme a tabela acima e colocá-los nesta pasta!
